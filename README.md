# md2pdf

Transform a markdown file (.md) into a pdf

# Requires
- pandoc

# Syntax
1.
```bash
pdfdoublepage.sh yourmd_file.md|.another_extension
```
will output yourmd\_file.pdf
2.
```bash
pdfdoublepage.sh yourmd_file.md another_name.pdf
```
will output another\_name.pdf
